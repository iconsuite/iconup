# iCONup v1.1

Utility desktop per l'installazione dei prodotti della **iCON Suite** via FTP/FTPS/SFTP.

## Funzionalità

- Upload via **FTPS** (TLS), **FTP** e **SFTP** (SSH)
- Progress bar in tempo reale
- Gestione profili multipli
- Crittografia AES-256 per i profili salvati
- Esporta/Importa profili
- Bilingue IT/EN

## Prodotti supportati

- iCONcms - Visual HTML Editor
- iCONstat - Analytics Dashboard
- iCONblog - Blog System
- iCONvert - HTML Converter
- Percorso personalizzato

## Come usare iCONup

1. **Scarica** l'app dalla pagina https://www.iconsuite.it/iconup/ e installala
2. **Scarica** il prodotto da installare (es. iCONstat, iCONcms...)
3. **Apri iCONup** 
4. **Inserisci i dati del server** — host, username e password del tuo hosting (li trovi nella mail di conferma di acquisto e/o nel pannello di controllo dell'hosting)
5. **Scegli il prodotto** da installare (es. iCONstat, iCONcms...)
6. **Seleziona la cartella** sul tuo computer contenente i file del software
7. **Clicca "Inizia Upload"** e attendi il completamento

Consiglio: **salva** i dati del server come profilo per riutilizzarlo in futuro.

## Troubleshooting

**Errori durante l'upload con FTPS**

Alcuni server FTP potrebbero non gestire correttamente il protocollo FTPS. In questo caso, seleziona FTP come protocollo alternativo.

## Sicurezza

Le credenziali sono salvate localmente in forma crittografata (AES-256-GCM). Non vengono mai inviate a server esterni.
